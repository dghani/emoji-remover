# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-10-11

### Added
- Initial release
- Multi-threaded emoji removal from files
- Support for processing individual files or directories
- Dry-run mode for previewing changes
- Gitignore support (enabled by default)
- Automatic trailing space removal after emojis
- Configurable file extensions
- Quiet mode for minimal output
- Pre-commit hook installation script
- Build-time emoji data embedding from Emoji-List-Unicode
